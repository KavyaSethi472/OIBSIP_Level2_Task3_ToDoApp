OIBSIP – Level 2 Task 3: To-Do Web App
About

This project is part of my Oasis Infobyte Web Development Internship (Level 2 – Task 3).
It is a To-Do List application that allows users to add, complete, delete, and filter tasks. The app also supports dark and light modes and saves data in the browser using local storage.

Features

Add, mark as complete, and delete tasks

Filter by all, completed, or pending tasks

Dark and light mode toggle

Tasks saved locally in the browser

Tech Stack

HTML5

CSS3

JavaScript

How to Run

Clone this repository:

git clone https://github.com/KavyaSethi472/OIBSIP_Level2_Task3_ToDoApp.git


Open the folder in VS Code.

Right-click on index.html and select Open with Live Server.
